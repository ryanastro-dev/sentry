# Generated Models

`models.ts` is generated by `ui/src-tauri/build.rs` from Rust structs in `ui/src-tauri/src/models.rs`.

This file is intentionally committed so frontend-only checks (`npm run check`, `npm run build`) can run without invoking Rust first.

Regenerate by running:

```powershell
cd ui/src-tauri
cargo check
```
